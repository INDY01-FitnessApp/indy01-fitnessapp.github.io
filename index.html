Indy-1 Fitness App

This website is an exploration of the development process that we underwent in putting together our senior project. We have been working on creating a fitness app that implements many of the usual functions of a fitness app while including a custom component that allows for a user to determine a starting and destination location that they might want to undertake and is capable of routing a path that will get a user from their chosen starting location to their destination location. Our app is then capable of using the user's data in order to track how far they are traveling and map that on top of the chosen route. This custom functionality along with some of the usual components of fitness apps are incorporated in the making of this project and this website contains the work that we completed in order to complete our development cycle.
